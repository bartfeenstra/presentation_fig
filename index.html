Getting us off the island: FIG

Framework Interoperability Group

The standards are open source

  • MIT (code)
  • Creative Commons (non-code)

The community is open source

  • #phpfig on irc.freenode.net.
  • https://groups.google.com/forum/?fromgroups#!forum/php-fig.

Voting members are PHP projects.

  • Members vote on whether proposals should be accepted as standards recommendations.
  • Drupal is a voting member. Its representative is Larry 'Crell' Garfield.
  • New members are approved by existing members.

Members are not required to implement any standards

  • Projects can disagree with standards.
  • Implementing new PSRs can break backwards compatibility (BC), which delays implementations.

PSR

PHP Standard Recommendation

  • PSR-x
  • All code can be included through Composer
  • All code and documentation is hosted on https://github.com/php-fig/fig-standards
